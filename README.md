GitHub Trending Repository Analysis

## Project Overview

This project collects data from GitHub Trending repositories using web scraping techniques and performs Exploratory Data Analysis (EDA) to identify trends in programming languages and popular repositories.

Objectives

* Scrape trending repositories from GitHub
* Create a custom dataset
* Analyze programming language popularity
* Visualize repository and language trends

Technologies Used

* Python
* BeautifulSoup
* Requests
* Pandas
* Matplotlib

Project Workflow

1. Collect trending repository data from GitHub
2. Store data in CSV format
3. Clean and preprocess data
4. Perform exploratory data analysis
5. Create visualizations

Features

* Web Scraping
* Data Collection
* Exploratory Data Analysis
* Data Visualization

Files

* scraper.py
* analysis.py
* visual1.py
* visual2.py
* github_trending.csv

Results

The analysis identifies the most popular programming languages and trending repositories on GitHub.

Future Improvements
